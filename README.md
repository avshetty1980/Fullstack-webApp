# Website with Node.js and Express


## Setting up the project

* In terminal, create directory `building-website-nodejs-express` and **change into it**.
* Run 
  ```bash
  git clone --bare git@github.com:danielkhan/building-website-nodejs-express.git .git
  git config --bool core.bare false
  git reset --hard
  git branch
  ```
  
Everything else will be discussed in my course.
